# Sudoku Solver

Solves n*n Grid where n>=3.


## Input

9 (grid size)

123456789 (list of symbols)

Board Values

.....9743

.5...8.1.

.1.......

8....5...

...8.4...

...3....6

.......7.

.3.5...8.

9724...5.

## Restrictions

* Only Grid with valid inputs will be allowed
* '.' should not be given in symbols list
